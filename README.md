## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18.x
- **npm** ≥ 9.x

### Installation

```bash
# Clone the repository
git clone https://github.com/Oms2004/Omiportfolio2.git

# Navigate to the project
cd Omiportfolio2

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```

## 🎨 Design Philosophy

- **Game-Engine Aesthetic**: Dark, immersive UI inspired by game engines and creative tools
- **Glassmorphism**: Frosted glass cards with subtle borders and backdrop blur
- **Gradient Accents**: Vibrant purple-to-cyan gradient used throughout
- **Micro-Animations**: Every section animates on scroll with staggered reveals
- **Typography**: Space Grotesk for headings, Inter for body — clean and modern
